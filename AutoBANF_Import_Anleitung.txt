
AUTOBANF EXCEL-IMPORT ANLEITUNG
==============================

VERWENDUNG:
-----------
1. Füllen Sie die Excel-Vorlage 'AutoBANF_Import_Template.xlsx' aus
2. Speichern Sie die Datei (Format .xlsx beibehalten)
3. Führen Sie das Import-Skript aus

WICHTIGE FELDER:
---------------
>> PFLICHTFELDER (muessen ausgefuellt werden):
- Artikelbeschreibung
- Artikelstueckpreis  
- Bestellmenge
- Waehrung (Standard: EUR)
- Steuerkennzeichen (Standard: kein separater Steuerabzug)
- Preisart (Standard: Netto-Preis)

>> OPTIONALE FELDER:
- Lange_Artikelbeschreibung
- Angebotsreferenz
- Angebotsdatum
- Lieferant
- Wunschlieferdatum
- Interne_Bemerkungen
- etc.

DATENFORMATE:
------------
- Preise: Dezimalzahl (12.50 oder 12,50)
- Datum: dd.MM.yyyy (25.01.2025)
- Datum-Bereich: dd.MM.yyyy - dd.MM.yyyy
- Menge: Ganze oder Dezimalzahl

DROPDOWN-WERTE:
--------------
Verwenden Sie nur die Werte aus dem 'Dropdown_Werte' Arbeitsblatt!

BEISPIEL-ZEILE:
--------------
Artikelbeschreibung: "USB-Kabel 2m"
Artikelstueckpreis: 15.99
Bestellmenge: 5
Waehrung: "EUR"
Lange_Artikelbeschreibung: "Hochwertiges USB-C Kabel, 2 Meter lang..."

IMPORT-VERHALTEN:
----------------
- Jede Zeile wird als separater Artikel angelegt
- Leere optionale Felder werden übersprungen  
- Bei Fehlern wird die Zeile übersprungen und protokolliert
- Screenshots werden für jede Zeile erstellt

FEHLERBEHEBUNG:
--------------
- Prüfen Sie Pflichtfelder
- Verwenden Sie korrekte Dropdown-Werte
- Prüfen Sie Datumsformate
- Achten Sie auf Sonderzeichen
